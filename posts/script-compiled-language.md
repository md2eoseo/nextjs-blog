# Script Language

- Python, JavaScript, Ruby, Perl, PHP ...
- 인터프리터로 코드를 한 줄 한 줄 읽고 실행한다.
- 런타임 중 에러가 발생할 가능성이 높다.

# Compiled Language

- C, C++, JAVA ...
- 컴파일러로 모든 코드를 기계어(byte code)로 변환시켜 단독으로 실행 가능한 파일로 만든다.
- 컴파일 중 에러를 검출할 수 있다.
- Windows에서 컴파일한 파일은 MacOS에서 실행할 수 없다.
